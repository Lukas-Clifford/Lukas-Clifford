### Lukas Clifford

Soy un desarrollador de software con un fuerte enfoque en el backend, aunque también me desenvuelvo en entornos full-stack. Mi filosofía se basa en construir software que sea a la vez robusto y eficiente, combinando una base técnica sólida con un enfoque creativo para la resolución de problemas.

Recién titulado en **Desarrollo de Aplicaciones Multiplataforma (DAM)**, he consolidado una base profunda en el ecosistema de Java. Mi entorno de trabajo habitual es Linux, y utilizo herramientas como Docker y Git de manera constante en mis proyectos. Mis lenguajes principales son Java y Python, y tengo experiencia con bases de datos relacionales como MariaDB, además de conocimientos en soluciones NoSQL.

Mi formación se ha centrado en la aplicación práctica, culminando en mi Proyecto Final de Grado: un **editor de música completo programado en Java**. Este proyecto me permitió manejar desde la arquitectura del software hasta el procesamiento de audio y la implementación de una interfaz funcional. Otros proyectos relevantes incluyen una aplicación de gestión de escritorio para un estudio de tatuajes, donde apliqué mis conocimientos de JavaFX, y el desarrollo de un backend en Java para el servidor de puntuaciones accedido desde una applicación Android.

Actualmente estoy buscando mi primera oportunidad laboral, un rol donde pueda contribuir, aplicar mis habilidades y seguir creciendo profesionalmente.

Si mi perfil te resulta interesante o quieres saber más sobre mis proyectos, puedes contactarme a través de mi email, **lukas.vogdt@gmail.com**, o en mi perfil de **[LinkedIn](http://linkedin.com/in/lukas-clifford-vogdt-torralba-ab33b9385)**.
